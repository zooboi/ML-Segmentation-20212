# Segmentation server

### Local testing
Require Python 3
- Download ```Nukki``` folder from ```https://drive.google.com/drive/folders/1rM5A1IXpXXK2OUT9r0CJIyYAg1YlMtPg```
- Install requirements:
    ```pip install -r requirements.txt```
- Run:
    ```python server.py```
- Socket server and UI now available at localhost:8000
